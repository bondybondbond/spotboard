# SpotBoard - Website Section Monitoring

[![Website](https://img.shields.io/badge/Website-Visit-blue)](https://bondybondbond.github.io/spotboard/)
[![Chrome Web Store](https://img.shields.io/badge/Chrome_Web_Store-Live-green)](https://chromewebstore.google.com/detail/spotboard/dnjleheimdlmdfajlciheclfbhmafiai?utm_source=github&utm_medium=readme&utm_campaign=badge)

Stop clicking through 4 sites daily. Track them all in one dashboard.

SpotBoard is a Chrome extension for monitoring website sections. Capture live widgets (trending deals, most read, scores, finance movers) into a personal dashboard with manual refresh.

## 🌐 Website
Visit [bondybondbond.github.io/spotboard](https://bondybondbond.github.io/spotboard/) for full documentation, privacy policy, and installation instructions.

## ✨ Key Features
- Unlimited captures (max 10/board)
- Manual refresh only (healthy tool relationship)
- Cross-device metadata sync (local-first, no backend)
- Smart exclusion mode (remove ads/timestamps)
- Sentiment coloring for finance data
- Position-based capture (rotating content)

## 🎯 Use Cases
- 📈 **Finance trackers**: Yahoo Finance movers, MarketWatch trends
- 💰 **Deal hunters**: HotUKDeals popular, Groupon daily deals
- 📰 **News readers**: BBC Most Read, Guardian top stories
- ⚽ **Sports fans**: ESPN scores, Premier League tables

## 🔒 Privacy First
- No accounts, no backend
- Captured HTML stays in your browser — never uploaded
- Component metadata syncs via Chrome sync
- Anonymous usage analytics via Google Analytics (no browsing history, no content)
- Zero tracking of which sites you visit outside SpotBoard

[Read full privacy policy](https://bondybondbond.github.io/spotboard/privacy)
